# Branding

Use:

- Reliability
- Preflight
- Regression testing
- User trust
- Public-facing QA
- Human-trivial checks

Avoid:

- Shame language
- Humiliation language
- Claims that a model is generally bad because it failed one obvious task

Primary tagline:

```text
Catch obvious AI mistakes before users do.
```
