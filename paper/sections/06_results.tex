\section{Results}

This section reports the frozen 2026-06-03 \paperSplit{} evidence snapshot.
The snapshot contains 223 model settings and 178 unique model identifiers. Each
row contains 224 scored attempts after the configured retry and repair policy;
provider errors, refusals, or timeouts are counted as incorrect final attempts
when they remain in the selected row.

Three model settings answered all 224 questions correctly, 80 reached at least
95\% answer correctness, and 40 scored below 80\%. These counts should be read
as first-draft benchmark evidence rather than a universal model ranking: the
split is intentionally small, public, and focused on short obvious tasks.

\paragraph{Deferred human validation.}
The fast preprint does not report measured human accuracy, response time
distributions, or model-versus-human gaps. A future strict version should add
participant count, no-tools instructions, exclusion policy, scoring method, and
a note that changing the item set invalidates reuse of collected human rows.
