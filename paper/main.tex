\begin{abstract}
\benchmarkname{} is a compact benchmark for short prompts that should be easy
for a careful human but still produce visible failures in public-facing language
models. The \paperSplit{} split contains 224 reviewed questions across character
counting, spelling transforms, small arithmetic, ordering, negation, format
compliance, word counting, and simple constraint awareness. Each item has an
objective target and a deterministic scorer. We report a frozen 2026-06-03
paper-v1 evidence snapshot covering 223 model settings and 178 unique model
identifiers. Three settings answered all 224 questions correctly, 80 reached
at least 95\% answer correctness, and 40 scored below 80\%. The strongest
aggregate failures were concentrated in character counting and spelling
transforms, while format-compliance failures often separated correct answers
from strict interface compliance. The fast preprint does not report a measured
human baseline; human-triviality is treated as an item-design target supported
by reviewed item cards, answer derivations, and scorer-gold artifacts.
\end{abstract}
